# 🔐 Password Strength Checker

## 📌 Project Description
This Python script checks the strength of a user-provided password by evaluating its:
- Length
- Use of uppercase/lowercase letters
- Numbers
- Special characters

The strength is categorized as:
- Very Weak
- Weak
- Moderate
- Strong
- Very Strong

## 🛠️ Tools & Technologies
- Python 3
- Regular Expressions
